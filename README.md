# 习思想复习闪卡 PWA

手机端离线闪卡复习应用，支持主观题、客观题、掌握度评分、本地学习记录和 PWA 添加到桌面。

## 使用

打开 GitHub Pages 地址后：

- iPhone：用 Safari 打开，点击分享按钮，选择“添加到主屏幕”。
- Android：用 Chrome 或 Edge 打开，选择“安装应用”或“添加到主屏幕”。

学习记录只保存在当前设备和当前浏览器中，不会上传服务器。

## 文件

- `index.html`：应用入口
- `data/cards.js`：应用读取的题库
- `data/flashcards-review.csv`：可审阅题库
- `docs/usage-guide.pdf`：使用说明
